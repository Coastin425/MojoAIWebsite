# MojoAI Website

Official website for MojoAI - Intelligent AI Solutions for Business

## Overview

This repository contains the MojoAI company website, designed to showcase our AI services and capabilities. The website is being developed in phases:

### Current Phase: Informational Website (v1)
The current version provides comprehensive information about MojoAI, including:
- Company overview and mission
- Service offerings (AI Workflows, Data Analytics, AI Integration)
- Contact information
- Placeholder for future login functionality

### Future Phases

#### MVP Phase (v2)

- User authentication and login

- Billing management

#### Advanced Phase (v3)
The advanced version will provide full AI tool access:
- Workflow management and automation

- Real-time analytics and reporting
- Advanced user features

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Coastin425/MojoAIWebsite.git
cd MojoAIWebsite
```

2. Open the website:
Simply open `index.html` in your web browser, or use a local server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve

# Or any other local server tool
```

3. Access the website:
Navigate to `http://localhost:8000` (or appropriate port) in your browser

## Customization

### Colors
Modify CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --dark-color: #1e293b;
    --light-color: #f8fafc;
    /* ... */
}
```

### Content
Update text content directly in `index.html` for:
- Company information
- Service descriptions
- Contact details
